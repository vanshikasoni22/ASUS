Core Workflows

📱 Retail Demo Application

Runs on in-store Chromebooks and tablets. Detects SKUs automatically, walks shoppers through specs, media, and features, and triggers an attract loop screensaver when idle.

💻 CMS Admin Portal

Desktop panel for ASUS operations managers. Allows file uploads, geographic target mapping (Regions, States, Cities, Stores), display distribution schedules, and syncing logs.

Design Tokens Statement: This design system represents the corporate look-and-feel of ASUS enterprise dashboards. It emphasizes clean layouts, Inter typography, 8px spacing grid, and ASUS Blue as the dominant brand identifier.

Design Guidelines

8-Point Grid

All paddings, margins, elevations, and layout heights align to sizes of 8px (8, 16, 24, 32, 48, 64).

Rounded Corners

Cards, inputs, dialog panels, and buttons use a consistent border radius of 8px. Outer chassis uses 16px.

Flat Shadow Scale

Clean elevation gradients matching Material Design 3. Low height shadows to create neat card separation.
